# Bully Bot
* A crowdsourced bot that fights against cyberbullying on twitter

## How to use it:

Slide into the bot's DMs and include the @ of someone who has cuberbullied you on twitter. The bot's twitter handle is [@bot_bully](https://twitter.com/bot_bully)

### Dependencies:

* **Python3:** python.org
* **Some python packages:** `pip install -r requirements.txt`
